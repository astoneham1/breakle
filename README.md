# Breakle

Breakle is a character guessing game. Your goal is to identify the secret Prison Break character in as few guesses as possible.

[Play now at https://astoneham1.github.io/breakle/](https://astoneham1.github.io/breakle/)

## How to Play

1. **Enter a Guess:** Start by typing the name of a *Prison Break* character in the search box.
2. **Review Clues:** Each guess reveals information about the secret character across six categories:
   - **Gender** (Male/Female)
   - **Nationality** (Represented by a flag)
   - **Status** (Alive/Deceased/Unknown)
   - **First Seen** (Season and Episode of first appearance)
   - **Relation** (Groups they belong to, such as "Sona Staff" or "Company")
   - **Episodes** (Total number of episodes they appeared in)
3. **Use Color Indicators:**
   - **Green:** Perfect match
   - **Yellow:** Close or partial match (e.g., same season, close episode count, or broad relation).
   - **Red:** No match.
4. **Track Progress:** Use the clue dots at the top to keep track of the correct attributes you've discovered.

## Features

- **Character List:** View a list of all available characters if you need a reminder.
- **Mobile Optimized:** A custom 3x2 grid layout for small screens ensures easy play on the go.
- **Accurate Stats:** All character data is as of the final episode (S5E9).
- **Interactive UI:** Smooth transitions and clear visual feedback for every guess.

## Built With

- **Static HTML/CSS/JS**
- **Twemoji** for consistent emoji rendering across all devices.
- **Static JSON** data for fast character searching and comparison.
