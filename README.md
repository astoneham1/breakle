# Breakle
Breakle is a character guessing game. Your goal is to identify the secret Prison Break character in as few guesses as possible.

Each guess provides clues based on the character's properties. Use these clues to narrow down your search! If you are stuck, you can choose to view a list of all possible characters to help ring a few bells.
