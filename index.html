Welcome to Breakle!

Breakle is a character guessing game. Your goal is to identify the secret Prison Break character in as few guesses as possible.

Each guess provides clues based on the character's properties. Use these clues to narrow down your search! If you are stuck, you can choose to view a list of all possible characters to help ring a few bells.

The character's status and stats are as of the final episode (S5E9), if you haven't watched all of Prison Break, don't play!

Color Indicators

Green: Perfect match! This property is exactly the same as the target character.

Yellow: Close or partial match. See the clue details below for what this means in each category.

Red: No match. The target character does not share this property.

Clue Categories

Gender
Matches if the character's gender is the same. No yellow match.
Nationality
Matches if the character's nationality is the same. Shown as a flag. No yellow match.
Status
Green if same status (Alive, Deceased). Yellow if target status is unknown but your guess is Alive or Deceased. Indicators at the top use abbreviations (A for Alive, D for Deceased).
First Seen
S0E0 format. Green if exact match. Yellow if same season. Arrows (↑/↓) show if target appeared earlier or later.
Relation
Green if they share specific groups. Yellow if they share broad archetypes. Examples of groups are being in a prison together, working on a team together, same family, etc. If there is one matching group, it will be shown. If there is multiple, it will be shown as "x2, x3", etc.
Episodes
Total episode count. Green if exact match. Yellow if within 10 episodes. Arrows show if target has more or fewer.
